# Contributing to WavyOS

## Prerequisites
1.  **Just:** Command runner (`brew install just`).
2.  **Podman:** Container engine.
3.  **BlueBuild CLI:** For local validation.

## Quick Start
*   `just build-vm`: Build the image locally (requires root).
*   `just test`: Build and run the VM.
*   `just lint`: **(TODO)** Run validation suite.
